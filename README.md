# ShareQR

## About
Share selected text or page URL as QR code

## Setup
##### 1. Clone repo
```
git clone https://github.com/JB1905/share-qr.git
```

##### 2. Go to directory
```
cd share-qr
```

## Build with
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [SCSS](https://sass-lang.com/)
